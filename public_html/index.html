Albuquerque Tool Shed: A peer to peer tool lending concept

The Idea: The cost of home maintenance, has never been higher. Projects require the use of specialized tools, purchased from the big box stores at considerable cost. Often, tools are only used a handful of times and languish in the garage, ultimately landing in the landfill.
"ABQ-Tool-Shed" is a peer to peer tool lending concept that connects community members to tools withing their communities, so that people can keep doing!

The Challenge: Build a mobile responsive site that successfully matches users with the tools they need, while building trust and confidence for users to lend and borrow safely.
